**Portafolio personal moderno y responsivo**, construido con React y Tailwind CSS.  
Diseñado para mostrar mi trayectoria como Analista de Sistemas y Desarrollador Full Stack, proyectos destacados, habilidades técnicas y certificaciones de forma interactiva.

🔗 **Demo en vivo:** [https://www.nicolascardinaux.com/](https://www.nicolascardinaux.com/)  
*(Reemplaza con tu enlace real una vez desplegado)*

- Escribe tu información en los archivos de la carpeta `data/`
- Ve los cambios reflejados al instante

## ✨ Features

- Diseño completamente responsivo para móviles, tablets y escritorio
- Animaciones fluidas con Framer Motion
- Efecto de escritura dinámica (React Typed)
- Tarjetas de proyectos y certificaciones interactivas (flip cards)
- Formulario de contacto funcional con FormSubmit (sin backend)
- Botón "Copiar email" con notificaciones (React Toastify)
- Soporte para modo oscuro/claro

## 🚀 Tech Stack

Este portafolio usa las siguientes tecnologías open source:

- [React.js] - Framework principal
- [Tailwind CSS] - Estilos utility-first
- [Framer Motion] - Animaciones y transiciones
- [React Typed] - Efecto de máquina de escribir
- [React Icons] - Iconos modernos
- [React Toastify] - Notificaciones elegantes
- [FormSubmit] - Envío de emails sin servidor

Y por supuesto, este portafolio es open source con un [repositorio público][repo] en GitHub.

## 📂 Estructura del Proyecto

```bash
src/
├── 📂 assets/       # Imágenes, logos y favicon
├── 📂 components/   # Componentes reutilizables
├── 📂 data/         # Datos del portafolio (proyectos, certificaciones, etc.)
├── 📂 lib/          # Utilidades y helpers
├── 📄 App.js        # Componente principal
└── 📄 index.js      # Entry point
```

## 💻 Instalación y Desarrollo Local

Requiere [Node.js](https://nodejs.org/) v14+ para ejecutarse.


# Clonar el repositorio
git clone https://github.com/NicolasCardinaux/nombre-de-tu-repo.git
cd nombre-de-tu-repo

# Instalar dependencias
npm install


# Iniciar servidor de desarrollo
npm start

Abre [http://localhost:3000](http://localhost:3000) para verlo en el navegador.
